== README

Community Base is a basic app that provides users the ability ask questions
and answers. App is written in Ruby on Rails and React.js.

Ruby version: 2.2.2+
Rails ver: 4.2.4

Basic start-up commands:
<tt>rake db:create
rake db:migrate
rake db:seed
rails s</tt>
