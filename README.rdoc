== README

Community Base is a basic app that provides users the ability ask questions
and answers. App is written in Rails and React.js.

Ruby version: 2.2.2+

Basic start-up commands:
<tt>rake db:create</tt>
<tt>rake db:migrate</tt>
<tt>rake db:seed</tt>
<tt>rails s</tt>
